Comunicação Client x Servidor

voltar

A comunicação Client x Servidor acontece quando um cliente (como um navegador) faz uma requisição a um servidor, que processa a informação e responde com os dados solicitados. Esse modelo é base para aplicações web, onde o cliente envia pedidos HTTP e o servidor responde com conteúdos como HTML, JSON ou arquivos. Essa troca ocorre de forma rápida e contínua durante o uso da aplicação.

Versionamento com Git e GitHub

voltar

Git é um sistema de controle de versão que permite acompanhar todas as alterações feitas no código. Com ele, é possível voltar a versões anteriores, trabalhar em paralelo com outras pessoas e manter a integridade do projeto. O GitHub entra como uma plataforma online para hospedar esses repositórios, facilitando a colaboração, o histórico de mudanças e o controle das versões de forma visual e acessível.

Estruturação e Formatação no HTML

O HTML é a base da web e serve para estruturar o conteúdo de uma página. Com ele, usamos tags de formatação como <strong>, <em>, <br> e <p> para organizar o texto e dar significado ao conteúdo.

Formulários

Formulários são essenciais para capturar dados do usuário. Com tags como <form>, <input>, <textarea>, <select> e <button>, é possível criar campos de login, cadastros e buscas. Esses dados geralmente são enviados ao servidor para processamento.

Mídias

O HTML permite inserir diferentes tipos de mídia com facilidade. Imagens usam a tag <img>, vídeos com <video> e áudios com <audio>. Também é possível usar <source> para diferentes formatos e garantir compatibilidade entre navegadores.

Tabelas

Para exibir dados organizados, usamos tabelas com <table>, <tr>, <td>, <th> e <thead>/strong<tbody>. Embora não sejam recomendadas para layout, são ótimas para representar informações tabulares de forma acessível.

Semântica

Tags semânticas como <header>, <nav>, <section>, <article>, <main> e <footer> ajudam a dar significado à estrutura da página. Elas melhoram a acessibilidade e o SEO, tornando o HTML mais legível tanto para humanos quanto para máquinas.
